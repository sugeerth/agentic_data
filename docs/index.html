Multi-Agent Architecture

A LangGraph-powered supervisor pattern with conditional routing and shared state management.

Supervisor (Orchestrator)
↓
Weather Agent (First - informs all others)
↓
Flight Agent
Hotel Agent
Activity Agent
↓
Budget Agent (needs flight/hotel prices)
↓
Itinerary Agent (Final Compilation)
# LangGraph Workflow Definition
from langgraph.graph import StateGraph, END

workflow = StateGraph(AgentState)
workflow.add_node("weather", weather_agent)
workflow.add_node("flight", flight_agent)
workflow.add_node("hotel", hotel_agent)
workflow.add_node("activity", activity_agent)
workflow.add_node("budget", budget_agent)
workflow.add_node("itinerary", itinerary_agent)

# Conditional routing based on completed agents
workflow.add_conditional_edges(
    "__start__", should_continue,
    {"weather": "weather", "flight": "flight", ...}
)
graph = workflow.compile()

Get Started in 30 Seconds

# Clone and setup
git clone https://github.com/sugeerth/agentic_data.git
cd agentic_data
pip install -r requirements.txt

# Set your API key
export OPENAI_API_KEY=your_key_here

# Launch the beautiful UI
streamlit run app.py

# Or use the CLI
python3 main.py -d "Tokyo" -b 3000 --dates "2025-06-01 to 2025-06-07"
